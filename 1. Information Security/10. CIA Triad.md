# 🛡️ CIA Triad, CVSS & Critical CVEs

## 📖 Introduction

The CIA Triad is the foundation of Information Security and helps organizations protect their assets through three core principles: Confidentiality, Integrity, and Availability. Along with the CIA Triad, vulnerability management frameworks such as CVSS and the understanding of critical CVEs play a vital role in modern cybersecurity.

---

# 🔐 CIA Triad

The CIA Triad represents the three fundamental objectives of information security.

## 1️⃣ Confidentiality

### 🎯 Objective

Prevent unauthorized access to sensitive information.

### 🔑 Security Controls

* Data Encryption (AES, TLS/SSL)
* Multi-Factor Authentication (MFA)
* Role-Based Access Control (RBAC)
* Network Segmentation
* Information Classification

### 💡 Real-World Examples

* Encrypting customer databases
* Restricting access to HR records
* Protecting cloud storage with MFA

---

## 2️⃣ Integrity

### 🎯 Objective

Ensure data remains accurate, consistent, and trustworthy.

### 🔑 Security Controls

* SHA-256 Hashing
* Digital Signatures
* Audit Logs
* Version Control
* Input Validation

### 💡 Real-World Examples

* Verifying downloaded files with checksums
* Digitally signing software releases
* Maintaining immutable audit trails

---

## 3️⃣ Availability

### 🎯 Objective

Ensure systems and data remain accessible when required.

### 🔑 Security Controls

* Backup Solutions
* Load Balancers
* Redundant Infrastructure
* DDoS Protection
* Disaster Recovery Planning

### 💡 Real-World Examples

* Cloud auto-scaling
* Database replication
* High Availability clusters

---

# 📊 CIA Triad Summary

| Principle       | Goal                        | Examples                    |
| --------------- | --------------------------- | --------------------------- |
| Confidentiality | Prevent unauthorized access | Encryption, MFA             |
| Integrity       | Maintain accurate data      | Hashing, Digital Signatures |
| Availability    | Ensure system accessibility | Backups, Redundancy         |

# 📈 Common Vulnerability Scoring System (CVSS)

## 📖 What is CVSS?

The Common Vulnerability Scoring System (CVSS) is an industry-standard framework used to assess and prioritize software vulnerabilities based on their severity.

---

## 🔍 CVSS Metrics

### Base Score

Measures the inherent severity of a vulnerability.

### Temporal Score

Accounts for factors that change over time such as exploit availability.

### Environmental Score

Adjusts severity based on the organization's environment.

---

## 📊 CVSS Severity Ratings

| Score Range | Severity |
| ----------- | -------- |
| 0.0         | None     |
| 0.1 - 3.9   | Low      |
| 4.0 - 6.9   | Medium   |
| 7.0 - 8.9   | High     |
| 9.0 - 10.0  | Critical |
