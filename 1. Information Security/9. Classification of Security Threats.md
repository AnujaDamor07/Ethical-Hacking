# 🛡️ Classification of Security Threats

Security threats can be classified into different categories based on their origin, target, and impact on information systems. Understanding these categories helps security professionals identify risks and implement appropriate defensive measures.

---

## 🌐 1. Network Attacks

### 📖 Overview

Network attacks target communication channels, protocols, and network devices to intercept, manipulate, or disrupt data transmission.

### 🔍 Common Types

| Attack Type              | Description                                              |
| ------------------------ | -------------------------------------------------------- |
| Eavesdropping            | Unauthorized interception of network traffic             |
| Man-in-the-Middle (MitM) | Intercepting and modifying communication between parties |
| DoS/DDoS                 | Flooding systems with traffic to disrupt services        |
| IP Spoofing              | Forging IP addresses to impersonate trusted systems      |
| Session Hijacking        | Taking control of an active user session                 |
| DNS Poisoning            | Redirecting users to malicious websites                  |

### 🛠️ Mitigation

* Use TLS/SSL encryption
* Deploy Firewalls and IDS/IPS
* Monitor network traffic continuously
* Implement network segmentation

---

## 💻 2. Host / Node Attacks

### 📖 Overview

These attacks focus on individual computers, servers, endpoints, or devices connected to a network.

### 🔍 Common Types

| Attack Type          | Description                                    |
| -------------------- | ---------------------------------------------- |
| Malware Infection    | Viruses, worms, trojans, ransomware            |
| Unauthorized Access  | Exploiting weak credentials or vulnerabilities |
| Privilege Escalation | Obtaining higher permissions                   |
| Rootkits             | Concealing malicious activities                |
| Data Exfiltration    | Stealing sensitive information                 |

### 🛠️ Mitigation

* Install Endpoint Detection & Response (EDR)
* Apply regular security patches
* Enforce strong authentication
* Follow the Principle of Least Privilege (PoLP)

---

## 🌍 3. Natural Threats

### 📖 Overview

Natural events can severely impact IT infrastructure and business operations.

### 🔍 Examples

* 🌊 Floods
* 🌎 Earthquakes
* 🔥 Fires
* ⚡ Lightning & Power Surges

### 🛠️ Mitigation

* Disaster Recovery Plans (DRP)
* Business Continuity Plans (BCP)
* Off-site backups
* UPS and power protection systems

---

## 🏢 4. Physical Threats

### 📖 Overview

Physical threats target hardware, facilities, and supporting infrastructure.

### 🔍 Examples

* Device theft
* Unauthorized facility access
* Hardware tampering
* Vandalism

### 🛠️ Mitigation

* CCTV surveillance
* Biometric access controls
* Security guards
* Physical security audits

---

## 👥 5. Human Threats

### 📖 Overview

Human threats arise from intentional or accidental actions performed by individuals.

### 🔍 Common Types

| Threat             | Description                               |
| ------------------ | ----------------------------------------- |
| Insider Threats    | Employees abusing authorized access       |
| Social Engineering | Manipulating users to reveal information  |
| Human Errors       | Misconfigurations and accidental mistakes |
| Sabotage           | Intentional damage to systems or data     |

### 🛠️ Mitigation

* Security awareness training
* Multi-Factor Authentication (MFA)
* Access monitoring
* Strong password policies

---

# 📊 Threat Classification Summary

| Category          | Primary Target          | Examples                  |
| ----------------- | ----------------------- | ------------------------- |
| Network Attacks   | Network Infrastructure  | MitM, DDoS, DNS Poisoning |
| Host/Node Attacks | Endpoints & Servers     | Malware, Rootkits         |
| Natural Threats   | Physical Infrastructure | Floods, Earthquakes       |
| Physical Threats  | Hardware & Facilities   | Theft, Tampering          |
| Human Threats     | People & Processes      | Phishing, Insider Threats |

---

## 🎯 Key Takeaway

Security threats originate from technical, physical, environmental, and human sources. A strong cybersecurity strategy requires a layered defense approach that combines technical controls, physical security, disaster recovery planning, and employee awareness programs.
