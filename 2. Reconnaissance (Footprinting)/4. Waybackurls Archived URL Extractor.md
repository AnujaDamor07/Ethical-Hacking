# 🕸️ Waybackurls – Historical URL Discovery Tool

**Waybackurls** is a lightweight reconnaissance utility created by **Tomnomnom** that retrieves archived URLs from the Internet Archive's Wayback Machine. It is widely used during **bug bounty hunting**, **penetration testing**, and **attack surface mapping** to identify historical endpoints, parameters, and resources that may no longer be visible on the live application.

---

## 🚀 Why Use Waybackurls?

* Discover historical URLs and endpoints
* Uncover forgotten or deprecated application functionality
* Collect URLs for content discovery and enumeration
* Support reconnaissance during security assessments
* Build target-specific URL wordlists for further testing

---

## 📦 Installation

### Install Golang

```bash
apt update
apt install golang -y
```

### Install Waybackurls

```bash
go install github.com/tomnomnom/waybackurls@latest
```

### Add Binary to PATH

```bash
cp -v /root/go/bin/waybackurls /usr/local/bin/
```

---

## 🛠️ Basic Commands

### Display Help Information

```bash
waybackurls -h
```

### Enumerate URLs for a Domain

```bash
waybackurls armourinfosec.com
```

### Using a Complete URL

```bash
waybackurls https://www.armourinfosec.com
```

### Process Multiple Domains

Create a target list:

```bash
vim domains.txt
```

Example:

```text
armourinfosec.com
thehackersworld.com
```

Run enumeration:

```bash
cat domains.txt | waybackurls
```

### Save Results

```bash
waybackurls tesla.com > tesla-urls.txt
```

---

## ⚙️ Useful Options

### Retrieve Archived Versions

```bash
cat domains.txt | waybackurls -get-versions
```

### Include Archive Timestamps

```bash
cat domains.txt | waybackurls -dates
```

### Ignore Subdomains

```bash
cat domains.txt | waybackurls -no-subs
```

### Export Results

```bash
cat domains.txt | waybackurls > domains-urls.txt
```

## 🔍 Data Processing & Cleanup

### Extract Unique Domains

```bash
cut -d '/' -f3 tesla-urls.txt | sort -u
```

### Remove Static Resources

```bash
grep -vE '\.(css|ico|svg|ttf|eot|woff|png|jpg|jpeg|gif)$' domains-urls.txt
```

### Strip Query Parameters

```bash
cut -d '?' -f1 domains-urls.txt
```

### Deduplicate URLs

```bash
grep -vE '\.(css|ico|svg|ttf|eot|woff|png|jpg|jpeg|gif)$' domains-urls.txt \
| cut -d '?' -f1 | sort -u
```

### Count Unique URLs

```bash
grep -vE '\.(css|ico|svg|ttf|eot|woff|png|jpg|jpeg|gif)$' domains-urls.txt \
| cut -d '?' -f1 | sort -u | wc -l
```

### Keep HTTPS URLs Only

```bash
grep '^https://' domains-urls.txt
```

---

## 🔗 Combining with HTTPX

Validate discovered URLs using HTTPX:

```bash
grep -vE '\.(css|ico|svg|ttf|eot|woff|png|jpg|jpeg|gif)$' domains-urls.txt \
| cut -d '?' -f1 | sort -u | grep '^https://' | httpx
```

### View Results

```bash
cat httpx-output.txt
```

### Remove Common Error Pages

```bash
grep -v "Page not found" httpx-output.txt
```
