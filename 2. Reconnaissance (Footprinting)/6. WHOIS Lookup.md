# 🌐 WHOIS Lookup

**WHOIS Lookup** is a process used to retrieve information about the ownership and registration details of a domain name (e.g., `example.com`).

---

## 📘 What is WHOIS Lookup?

WHOIS is a protocol and database system used to store domain registration information. A WHOIS lookup can provide details such as:

* Domain owner information (if not protected by privacy services)
* Contact email addresses and phone numbers
* Domain creation and expiration dates
* Domain registrar information
* Name servers (DNS configuration)
* Domain status codes (e.g., `clientTransferProhibited`)

### Command-Line Usage (Linux)

Basic lookup:

```bash
whois example.com
```

Check multiple domains:

```bash
for domain in example.com test.com domain.com; do
    whois $domain
done
```

Save output to a file:

```bash
whois example.com > example-whois.txt
```

Query a specific WHOIS server:

```bash
whois -h whois.nic.xyz example.xyz
```

## ⚠️ Important Considerations

* **Privacy Protection:** Many registrars offer privacy services that hide registrant information.
* **Data Accuracy:** WHOIS information may be outdated or intentionally obscured.
* **Rate Limiting:** Excessive queries may trigger temporary restrictions.
* **GDPR Compliance:** Personal information may be redacted for privacy reasons.
* **Automation:** When performing large-scale lookups, follow provider policies and rate limits.
