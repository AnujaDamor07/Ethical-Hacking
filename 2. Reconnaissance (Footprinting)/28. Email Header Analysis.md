## вҡҷпёҸ How Email Tracking Works

When an email is sent, each mail server it passes through adds a `Received:` header to the email metadata.

By examining these headers, investigators can:

* вң… Trace the **origin IP address**
* рҹ§ӯ View **timestamp logs** of mail hops
* рҹҡЁ Detect **header forgery** or **spoofing**

---

* `mail.example.com` вҖ“ SenderвҖҷs mail server

## рҹ§ҫ How to View Email Headers (with Download Method вҖ“ Offline)

### вң… Step-by-Step (Gmail)

1. Open the email.
2. Click the **3-dot menu (вӢ®)** near the reply button.
3. Select **"Download message"** вҖ” this saves a file with `.eml` extension.
4. Locate the `.eml` file in your **Downloads** folder.
5. **Right-click вҶ’ Open with вҶ’ Notepad** (or any text editor).

---

### рҹ“Ӯ Structure of the `.eml` or Raw Email File

When you open the downloaded file:

* Everything **above the first empty line** is the **email header**.
* Everything **after the empty line** is the **email body** (actual content).

Example:

```text
Received: from sender.domain.com (123.45.67.89)
Date: Tue, 2 Jul 2025 10:20:00 +0000
From: attacker@example.com
To: victim@example.com
Subject: Urgent Invoice

Hello, please check the attached invoice...
```

рҹ”Қ In this case:

* Header ends before the **empty line**
* Body starts **after the empty line**

---

### рҹ–ҘпёҸ Outlook (Desktop App)
