# 🗡️ Katana – Advanced Web Crawling Framework

**Katana** is a fast and modern web crawling framework developed by **ProjectDiscovery**. It is widely used by **security researchers, bug bounty hunters, and penetration testers** to discover web assets, endpoints, parameters, and hidden application content during reconnaissance.

---

## 🚀 Core Features

### ⚡ High-Speed Crawling

Designed with modern concurrency techniques, enabling efficient crawling of large and complex web applications.

### 🧩 Extensible Architecture

Supports multiple reconnaissance and discovery workflows, including:

* File & directory discovery
* Parameter collection
* Endpoint enumeration
* Custom module integration

### 🌐 JavaScript Processing

Capable of analyzing JavaScript-heavy applications and Single Page Applications (SPAs), helping uncover hidden routes, endpoints, and parameters.

### 🔗 Toolchain Integration

Integrates smoothly with other ProjectDiscovery tools such as:

* Nuclei
* HTTPX
* Naabu

This allows seamless automation of reconnaissance and vulnerability assessment workflows.

### 🎛️ Flexible Configuration

Provides granular control over crawling scope, depth, filtering options, and output formatting.

## 📥 Installation

### Install Golang

```bash
apt update
apt install golang -y
```

### Install Katana

```bash
go install github.com/projectdiscovery/katana/cmd/katana@latest
```

---

## 🛠️ Usage Examples

### Display Help Menu

```bash
katana -h
```

### Basic Website Crawl

```bash
katana -u https://uwu.armourinfosec.com
```

### Save Results to a File

```bash
katana -u https://uwu.armourinfosec.com -o katana-output.txt
```

### Enable JavaScript Crawling

```bash
katana -u https://uwu.armourinfosec.com -jc
```

### Set Crawling Depth

```bash
katana -u https://uwu.armourinfosec.com -d 3
```

---

## 📊 Output Processing

### Exclude Specific Domain Entries

```bash
grep -v "uwa\.armourinfosec\.com" katana-output.txt
```

### Extract Unique JavaScript Files

```bash
grep '\.js' katana-output.txt | cut -d '?' -f1 | sort -u
```
