# рҹ•өпёҸвҖҚвҷӮпёҸ What is an IP Logger?

An **IP Logger** is a script or tool used to capture and record the IP address of visitors accessing a specific link, file, or page.

## рҹ§  What It Logs

When someone visits the page, the logger typically collects:

* рҹҢҗ **IP Address**
* рҹ•’ **Timestamp**
* рҹ–ҘпёҸ **Browser & OS (User-Agent)**
* рҹ”— **Referrer**
* рҹ“„ **Page Visited**

### рҹ§‘вҖҚрҹ’» `index.html` вҖ“ Requests GPS & Sends to Backend

```html
<!DOCTYPE html>
<html>
<head>
    <title>GPS IP Logger</title>
</head>
<body>
    <h2>Getting your GPS location...</h2>
    <p id="status">Waiting for user permission...</p>

    <script>
        if ('geolocation' in navigator) {
            navigator.geolocation.getCurrentPosition(
                function(position) {
                    const lat = position.coords.latitude;
                    const lon = position.coords.longitude;

                    fetch('gps-logger.php', {
                        method: 'POST',
                        headers: {
                            'Content-Type': 'application/x-www-form-urlencoded'
                        },
                        body: `lat=${lat}&lon=${lon}`
                    });

                    document.getElementById('status').innerHTML = `Location sent: ${lat}, ${lon}`;
                },
                function(error) {
                    document.getElementById('status').innerHTML = `Error: ${error.message}`;
                }
            );
        } else {
            document.getElementById('status').innerHTML = "Geolocation not supported by your browser.";
        }
    </script>
</body>
</html>

```

### рҹ§ҫ `gps-logger.php` вҖ“ Logs GPS + IP

```php
<?php
if ($_SERVER['REQUEST_METHOD'] !== 'POST') {
    exit("No GPS data received.");
}

date_default_timezone_set('Asia/Kolkata');

// Sanitize input
$lat = $_POST['lat'] ?? 'Not provided';
$lon = $_POST['lon'] ?? 'Not provided';
$ip = $_SERVER['REMOTE_ADDR'];
$ua = $_SERVER['HTTP_USER_AGENT'] ?? 'Unknown';
$time = date("Y-m-d H:i:s T");

// Generate Google Maps link placeholder
$mapLink = "https://www.google.com/maps?q={$lat},{$lon}";

// Prepare structured log entry
$log = <<<LOG
<hr>
<b>Time:</b> $time<br>
<b>IP:</b> $ip<br>
<b>Latitude:</b> $lat<br>
<b>Longitude:</b> $lon<br>
<b>Google Maps:</b> <a href="$mapLink" target="_blank">View Location</a><br>
<b>User-Agent:</b> $ua<br>
LOG;

// Append tracking data safely to the log file
file_put_contents('gps-log.html', $log . PHP_EOL, FILE_APPEND | LOCK_EX);
?>

```
