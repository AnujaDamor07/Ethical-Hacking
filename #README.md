🧠 Information Security (InfoSec)

Basic overview of protecting data, systems, and networks from unauthorized access and threats.

🔐 Cyber Security

Protection of digital systems, networks, and applications from cyber attacks.

🕳️ Vulnerability

A weakness in a system that can be exploited by attackers.

🧪 Penetration Testing (Pentesting)

Simulated cyber attack to find security weaknesses before attackers do.

🗂️ Penetration Testing Methodologies

Structured approaches like OWASP, PTES, NIST used in testing.

🌀 Phases of Penetration Testing & Hacking Lifecycle

Recon → Scanning → Gaining Access → Maintaining Access → Reporting.

🎯 Scope of Penetration Testing

Defines what systems, networks, or apps are allowed to be tested.

🪲 Bug Bounty

Programs where companies reward security researchers for finding bugs.

🟥🟦🟪 Red Team / Blue Team / Purple Team
Red: Attack simulation
Blue: Defense
Purple: Collaboration
⚠️ Security Threats

Possible dangers that can harm systems or data.

📊 Classification of Security Threats

Includes network, host, physical, and human-based threats.

🔺 CIA Triad

Confidentiality, Integrity, Availability — core security principles.

🛡️ Defense-in-Depth (DiD)

Multiple security layers to protect systems.

👨‍💼 Chief Information Security Officer (CISO)

Senior role responsible for security strategy in an organization.

🧩 Vulnerability Management

Process of identifying and fixing security weaknesses.

🌐 Vulnerability Research Websites

Sources like NVD, Exploit DB, CVE Details, etc.

🔬 Vulnerability Research

Finding and analyzing security flaws in systems.

🇮🇳 CERT-In (Indian Computer Emergency Response Team)

India’s national cybersecurity incident response organization.
